# Bootcamp React do Zero + Tailwind

Repo único do bootcamp. O app React roda em `./app`.

## Requisitos

- Node: 24 (ver `.nvmrc`)
- Package manager: pnpm

## Como rodar

```bash
cd app
pnpm install
pnpm dev
```

## Scripts (app)

```bash
cd app
pnpm dev
pnpm build
pnpm lint
```

## Estrutura

- `core.md` — regras do bootcamp
- `bootcamp_status.json` — estado atual (fonte de verdade)
- `docs/decisions.md` — decisões permanentes
- `docs/daily/` — logs diários
- `app/` — projeto Vite + React + Tailwind
